# Movie Recommendation System

This project is a simple movie recommendation system built using Python.

It uses movie ratings and vote counts to find the top movies using IMDb's weighted rating formula.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Files
- movie_recommender.ipynb
- requirements.txt
- movies_metadata.csv

## How to Run
1. Open the notebook in Google Colab
2. Upload the dataset file
3. Run all cells

## Output
The project shows the top-rated movies based on weighted score.
